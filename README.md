# workshop-ui-ux-26

aplicativo: Microsoft Autenticator

Problema: Nunca aparece o codigo de verificação quando voce solicita para entrar na conta instituicional, assim, meio que obrigando o usuario tentar logar de outras formas, como por sms

Porque deve ser melhorado: facilitar a autenticação do usuario para adentrar em sua conta institucional

Solução: Sempre que solicitado o codigo pelo app, aparecer uma notificação de tentativa de login e assim que voce entrar no aplicativo, aparecer um pop-up do numero de verificação ou até mesmo usar a biometria para a autenticação
